Use OkHttpClient.Builder().connectTimeout(10, TimeUnit.SECONDS) and OkHttpClient.Builder().readTimeout(10, TimeUnit.SECONDS) or writeTimeout(10, TimeUnit.SECONDS) to set connection and read/write timeouts.
